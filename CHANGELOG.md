# Changelog

## [1.0] - 2026-04-02

### Initial release
- Random mouse jitter every 2–5 seconds
- Ctrl+Tab (or Cmd+Tab on macOS) window cycling every 45–90 seconds
- Random fake keystrokes every 8–20 seconds (togglable)
- OK button toggles Mac/Windows mode
- Right button toggles fake typing on/off
- Hold Back to exit and restore USB
